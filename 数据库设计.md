# 毕业设计之数据库设计

毕业设计的名字：滋养果园

英文名：Nourish Orchard

项目名称：NourishOrchard

## 账户表

| 字段名     | 类型   | 描述     |
| ---------- | ------ | -------- |
| username   | string | 账号名   |
| password   | string | 密码     |
| promise    | int    | 用户权限 |
| created_at | date   | 创建时间 |
| updated_at | date   | 更新时间 |
| deleted_at | date   | 删除时间 |

主键：username，要带有公司标识，唯一

## 用户表

| 字段名   | 类型   | 描述                 |
| -------- | ------ | -------------------- |
| id       | bigint | 用户主键ID           |
| username | string | 账号名，外键->账户表 |
| name     | string | 用户实名             |
| gender   | string | 性别                 |
| phone    | string | 联系电话             |
| address  | string | 家庭地址             |
| birthday | date   | 生日                 |
